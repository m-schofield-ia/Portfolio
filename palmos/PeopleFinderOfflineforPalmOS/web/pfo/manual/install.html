Installation

Installation of PeopleFinder Offline for Palm OS is much like installation of any other Microsoft Windows program:

  1. Double click the PFOforPalmOS-X.Y.exe file.
  2. Select Next a couple of times.
  3. Let the installer do its work - during installation you may be prompted to select a Palm Profile to receive the PFO.prc Palm program. You will only be prompted if you have multiple HotSync IDs on your computer.


Once finished, you'll find a PeopleFinder Offline for Palm OS sub folder in your Programs folder of the Start menu. The sub folder contains the following entries:

  • ChangeLog - a simple Log describing the changes done to PeopleFinder Offline for Palm OS during its lifetime.
  • Converter - a program to convert the PeopleFinder Offline database to Palm OS databases. See the Conversion Guide for more information.
  • PFO Installer - a program to queue the PFO.prc Palm program for HotSync.
