Requirements

The following requirements must be met before installing and using PeopleFinder Offline for Palm OS:

  • You must be running Microsoft Windows 2000 or better. PeopleFinder Offline for Palm OS is developed and tested on Microsoft Windows XP Service Pack 2.
  • You must have Microsoft .NET Framework 2.0 or better installed. Go to Microsoft .NET Framework downloads page and select the ".NET Framework Version 2.0 Redistributable Package (x86) link. The direct link to the software download is http://www.microsoft.com/downloads/details.aspx?FamilyID=0856eacb-4362-4b0d-8edd-aab15c5e04f5&DisplayLang=en (ugly link by the way!)
  • You must have PeopleFinder Offline installed. You must install the full package - both the PC and the PDA options.
  • You must have a Palm OS 4.0 (or better) device.
