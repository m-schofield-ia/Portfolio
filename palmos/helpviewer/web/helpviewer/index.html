Plua Help Viewer v1.0

Plua Help Viewer is a small utility to read the topics in the Plua Help files (PluaHelp.prc and plua2help.prc) from the Plua 1 or Plua 2 distributions.

If you would like to localize Plua Help Viewer to another language, Drop me a mail.

Plua Help Viewer runs on any Palm OS 3.1 or better device.
