@Tracker - Web Export Instructions

To use the @Tracker Web Export tool you must have a @Tracker archive at hand. Instructions on how to create a archive is documented in the @Tracker User Manual. HotSync the archives to your desktop computer and then follow these steps:

  1. Click the Browse button and locate the archive file. The file will be stored in the backup folder in your HotSync hierarchy. The archive file will be named as TTTX_{archive name}.pdb.
  2. Select the desired date and time formats. This can be set to match the setting on your handheld.
  3. Select the Linefeed type. Depending on your CSV reader/Operating System it may matter what kind of Linefeeds you use.
  4. Select the output format - should your archive file be exported to Excel CSV, Standard CSV, PHP or debug formats?
  5. Leave the Keep Whitespaces checkbox unchecked if you want to have white spaces characters such as tabulators converted to a space.
  6. When satisfied with your settings click on the Export button and your file will be uploaded, exported and offered as a download.

The @Tracker Web Export tool has been tested using Mozilla Firefox. It should work using other browsers. If not, I would like to hear from you.
