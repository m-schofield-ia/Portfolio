@Tracker v2.0

@Tracker is a small Activity Time Tracker used to record time spent in various tasks. The target audience are users in need of tracking time spent in f.ex. project related tasks in their daily job.

@Tracker runs on any Palm OS 3.1 or better device.

@Tracker is released under the GNU GPL 2.

tttxexport

tttxexport is a tool used to export the archives generated by @Tracker to f.ex. Excel CSV or Standard CSV:

  • @Tracker Web Export - export archive files to CSV etc. using this online tool.
  • tttxexport-1r4 - for Windows. Please read the ReadMe after installation.
  • TTTXExportGui-1r1 - for Windows. This is a simple wrapper GUI around tttxexport. You need to have tttxexport installed before installing TTTXExportGUI. Please read the ReadMe after installation.
  • tttxexport-1r4 - the sources. If you want to build tttxexport your self. tttxexport is known to build in Linux and under Microsoft Visual Studio .NET 2003 (Windows).
