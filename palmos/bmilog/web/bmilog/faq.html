What exactly is Body Mass Index?
The Body Mass Index (BMI) is an objective measure that can help you decide whether your weight is right for you or not. BMI is a figure calculated from your height and weight and can be used to determine if you are overweight, underweight or at a healthy weight.
Cool! So how is that magic number calculated?
Depends ... if you are using the metric system then it is weight / height * height (height is in meters)
... and?
If you are using pounds and inches then it is: weight * 704.5 / height * height (height is in inches)
So ... what is it good for?
If your BMI checks out in the range 20 - 24.9 you're at a healthy weight. Below 20 you're underweight. 25 or higher, you're overweight.
